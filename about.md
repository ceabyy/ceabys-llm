## Projects

**HappyWoof** - An AI-powered dog emotion interpreter that was built by finetuning ResNet18. The backend utilises Flask and the frontend uses Vanilla JS, HTML and CSS. The website was deployed on Github Pages and the model is deployed on Hugging Face Spaces. This project was inspired by the fact that many of my friends have dogs and just other pets in general. The website link can be found on https://ceabyy.github.io/HappyWoof/. Just do note that the model will take about a minute or two to start running if it hasn't been used in a while (which it probably hasn't).

**Portfolio Website** - My personal portfolio website that was built using React, Vite, Typescript, Tailwind CSS, HTML and Framer Motion. The website is deployed using Vercel.

**Ceaby's LLM** - This AI bot that answers questions about me. This was implemented by using Supabase to store the chunk embeddings and its vectors. Then, a pretrained model is used with RAG to answer these questions accurately off of the information in the database.

**Adaptive Pomodoro Timer** - This project is not finished, and the AI model has not been fully implemented yet, but it uses linear regression at the moment as a placeholder. It's meant to be an AI-powered pomodoro timer that finetunes the ratios between study and break periods based off of ratings made after each session. This was inspired by my profuse usage of the pomodoro method during secondary school.

**Investigating the Welfare of Remote Workers** - A data analysis and science project that mainly uses Tableau and a machine learning model implemented with Python to gather feature importances to better understand the nature of the data. The result is a Tableau dashboard that will soon be implemented on a website.
